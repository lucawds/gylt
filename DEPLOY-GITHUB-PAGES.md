# GYLT kostenlos mit GitHub Pages veröffentlichen

## 1. Repository erstellen

1. Öffne `https://github.com/new`.
2. Gib dem Repository einen Namen, zum Beispiel `gylt`.
3. Stelle es auf `Public`.
4. Klicke auf `Create repository`.

## 2. Dateien hochladen

1. Öffne dein neues Repository.
2. Klicke auf `Add file` und dann `Upload files`.
3. Ziehe alle Dateien aus diesem Ordner hinein:
   `C:\Users\furzl\homepage`
4. Wichtig: Der Ordner `pictures` muss mit hochgeladen werden.
5. Klicke unten auf `Commit changes`.

## 3. GitHub Pages aktivieren

1. Öffne im Repository `Settings`.
2. Links auf `Pages`.
3. Bei `Build and deployment` wählst du:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - Ordner: `/root`
4. Klicke auf `Save`.

## 4. Link öffnen

Nach ungefähr 1 bis 3 Minuten zeigt GitHub dir den Link an, meistens so:

```text
https://deinname.github.io/gylt/
```

Öffne den Link auf deinem Handy.

## 5. Auf den Homescreen legen

Auf iPhone:

1. Link in Safari öffnen.
2. Teilen-Symbol drücken.
3. `Zum Home-Bildschirm` wählen.

Auf Android:

1. Link in Chrome öffnen.
2. Menü öffnen.
3. `Zum Startbildschirm hinzufügen` oder `App installieren` wählen.
